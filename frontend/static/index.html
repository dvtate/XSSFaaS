XSSaaS

Distributed, serverless cloud that's actually just a bunch of browser tabs

Yet another serverless FaaS platform except this one runs your functions in random people's browsers. Or alternatively you could self-host and use, for example, your own computer lab.

Technology In Use

XSSaaS uses modern web technology in order to maximize performance and ease of use while avoiding the bloat of many popular frameworks.

  • Web Workers
  • WebSockets
  • HTML + TypeScript + Webpack (no frameworks)

Stack

In order to leverage the platform you may find some recent web technologies useful. For example, WebAssembly - in order to get optimal performance or use pre-existing code and WebRTC - for communication between workers.

How it works

This looks like a lot but really it's not.

  • Upload JavaScript file with default function export (along with any other relevant project files)
  • Specify any other policies and metadata assocaited with the function
  • Open a worker tab (or rely on others to donate their resources)
  • Use an API endpoint to send tasks to the function (which can include relevant data)

Simply upload a javascript file with a default export that's callable and any other relevant project files, specify any relevant policies for it. Open a worker tab (or rely on others to generously donate their resources).
